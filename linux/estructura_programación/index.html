Chuleta de Bash

[ condición ] && acción_si_true || acción_si_false
Ejemplo: 
[ $? -eq 0 ] && echo "Operación exitosa" || echo "Operación fallida"

Operador Descripción Ejemplo
-eq Igualdad numérica [ 5 -eq 5 ]
-ne No igual numérico [ 5 -ne 3 ]
-gt Mayor que [ 5 -gt 3 ]
-lt Menor que [ 3 -lt 5 ]
-ge Mayor o igual que [ 5 -ge 5 ]
-le Menor o igual que [ 3 -le 5 ]
== Igualdad de cadenas [ "foo" == "foo" ]
!= No igualdad de cadenas [ "foo" != "bar" ]
-z Cadena vacía [ -z "" ]
-n Cadena no vacía [ -n "foo" ]
&& Lógico AND [ 5 -gt 3 ] && [ 3 -lt 5 ]
|| Lógico OR [ 5 -gt 3 ] || [ 3 -gt 5 ]
! Lógico NOT ![ 5 -lt 3 ]

variable=$(condición && echo "valor_true" || echo "valor_false")
Ejemplo: 
mensaje=$( [ -f archivo.txt ] && echo "Existe" || echo "No existe" )

comando && echo "Éxito" || echo "Fallo"
Ejemplo: 
rm archivo.txt && echo "Archivo eliminado" || echo "Error al eliminar archivo"

condición && comando
Ejemplo: 
[ -d carpeta ] && cd carpeta

función() { comando; }
Ejemplo: 
saludar() { echo "Hola, $1"; }; saludar "Mundo"

for i in {1..5}; do comando; done
Ejemplo: 
for i in {1..5}; do echo "Iteración $i"; done

case variable in patrón1) comando;; patrón2) comando;; esac
Ejemplo: 
case $1 in
    start) echo "Iniciando...";;
    stop) echo "Deteniendo...";;
    *) echo "Opción no válida";;
esac

while condición; do comando; done
Ejemplo: 
contador=0
while [ $contador -lt 5 ]; do
    echo "Iteración $contador"
    contador=$((contador+1))
done

Ejercicios de Bash

Enunciado: Crea un script que reciba un número como argumento y determine si es par o impar. Solución: 

#!/bin/bash
if [ $(($1 % 2)) -eq 0 ]; then
    echo "El número $1 es par."
else
    echo "El número $1 es impar."
fi

Enunciado: Crea un script que reciba el nombre de un archivo y un directorio, y compruebe si el archivo existe. Solución: 

#!/bin/bash
if [ -f "$2/$1" ]; then
    echo "El archivo $1 existe en el directorio $2."
else
    echo "El archivo $1 no se encuentra en el directorio $2."
fi

Enunciado: Crea un script que muestre los 5 procesos que más memoria consumen. Solución: 

#!/bin/bash
echo "Top 5 procesos que más memoria consumen:"
ps -eo pid,comm,%mem --sort=-%mem | head -n 6

Enunciado: Crea un script que reciba un archivo y cuente el número de líneas, palabras y caracteres. Solución: 

#!/bin/bash
if [ -f "$1" ]; then
    lineas=$(wc -l < "$1")
    palabras=$(wc -w < "$1")
    caracteres=$(wc -m < "$1")
    echo "Líneas: $lineas"
    echo "Palabras: $palabras"
    echo "Caracteres: $caracteres"
else
    echo "El archivo $1 no existe."
fi

Enunciado: Crea un script que realice una copia de seguridad de un directorio. Solución: 

#!/bin/bash
origen=$1
destino=$2/backup_$(date +%Y%m%d)

if [ -d "$origen" ]; then
    mkdir -p "$destino"
    cp -r "$origen"/* "$destino"
    echo "Copia de seguridad completada en $destino."
else
    echo "El directorio $origen no existe."
fi
